# Curriculum Design

Turn learning goals into complete, structured curricula in minutes. This skill works for K-12 educators, university instructors, corporate trainers, and instructional designers.

## How It Works

```
┌─────────────────────────────────────────────────────────────────┐
│ CURRICULUM DESIGN                                               │
├─────────────────────────────────────────────────────────────────┤
│ ALWAYS (works standalone)                                       │
│ ✓ You tell me: subject, level, duration, learning goals        │
│ ✓ Build: scope and sequence, unit plans, lesson outlines       │
│ ✓ Align: learning objectives to Bloom's taxonomy              │
│ ✓ Output: complete curriculum document with assessments        │
├─────────────────────────────────────────────────────────────────┤
│ SUPERCHARGED (when you connect your tools)                      │
│ + LMS: auto-publish to Canvas, Google Classroom               │
│ + Notion: store curriculum in your knowledge base             │
│ + Standards databases: align to Common Core, NGSS, etc.       │
└─────────────────────────────────────────────────────────────────┘
```

## Getting Started

To trigger this skill, say something like:
- "Design a 6-week curriculum on climate change for 9th graders"
- "Create a Python programming course for beginners, 12 hours total"
- "Build a unit plan on the French Revolution for AP History"

**Required:**
- Subject or topic
- Target audience (grade level, experience level, role)
- Duration (weeks, hours, sessions)

**Helpful if you have it:**
- Specific learning standards to align to
- Existing materials or textbooks to reference
- Assessment constraints (no high-stakes tests, project-based only, etc.)

## Execution Flow

### Step 1: Gather Requirements
If not provided, ask:
- "What subject and topic?"
- "Who is the audience? (grade level, background, experience)"
- "How long is the course? (weeks, hours, number of sessions)"
- "Any specific standards or outcomes to hit?"
- "What assessment types are allowed/preferred?"

### Step 2: Research & Structure
1. Identify prerequisite knowledge and entry points
2. Map learning progression from foundational to complex
3. Apply Bloom's taxonomy to write measurable objectives
4. Design backward from summative assessment

### Step 3: Generate Curriculum
1. Write scope and sequence overview
2. Develop unit plans with essential questions
3. Outline individual lessons at the right grain size
4. Design aligned assessments for each unit
5. Identify differentiation strategies

### Step 4: Review & Refine
- Check: Are objectives measurable and achievable?
- Check: Does the sequence build logically?
- Check: Are assessments aligned to objectives?
- Ask: "Want me to expand any unit or develop full lesson plans?"

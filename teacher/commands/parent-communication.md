# Parent Communication

> If you see unfamiliar placeholders or need to check which tools are connected, see [CONNECTORS.md](../CONNECTORS.md).

Draft positive, professional communications to families including updates, celebrations, concerns, and meeting requests.

## Usage

```
/parent-communication <topic> <audience> <details>
```

Examples:
- `/parent-communication celebration elementary "Student excelled in science project"`
- `/parent-communication concern high-school "Student assignment submission has decreased"`
- `/parent-communication conference request any "Request for parent-teacher conference"`

## Communication Types

| Type | Purpose |
|------|---------|
| `celebration` | Share positive news and successes |
| `concern` | Address academic or behavioral concerns |
| `update` | Provide routine progress updates |
| `request` | Request meeting or information |
| `invitation` | Invite families to events |

## Workflow

### 1. Determine Communication Type

Identify:
- **Purpose**: What do you want to achieve?
- **Audience**: Parent(s), guardian(s), family as a whole
- **Tone**: Positive, concerned, neutral, urgent
- **Medium**: Email, letter, message, phone script

### 2. Structure the Message

#### Header
- Professional greeting
- Your name and role
- Student name

#### Body
- **Opening**: Positive or purpose statement
- **Context**: Background information
- **Details**: Specifics of the situation
- **Action**: What you're asking for or offering

#### Closing
- Reaffirm partnership
- Contact information
- Appreciation

### 3. Apply Communication Principles

**For Celebrations**
- Be specific about what was accomplished
- Include how the student achieved it
- Express genuine enthusiasm
- Consider public vs. private recognition

**For Concerns**
- Use "I notice" language
- Be specific and factual
- Include context and history
- Focus on solutions and partnership
- Follow school protocol for documentation

**For Requests**
- Be clear about what you're requesting
- Provide context and purpose
- Offer flexibility when possible
- Include timeline

### 4. Draft Template

```
[Subject Line: Clear and concise]

Dear [Parent/Guardian Name(s)],

[Opening - positive or purpose statement]

[Context and details]

[I notice/From our observations/In our class...]

[Specific examples and circumstances]

[What we're doing/What we're asking]

[Next steps and collaboration]

Thank you for your partnership in [Student Name]'s education. Please don't hesitate to reach out if you have questions or concerns.

Best regards,
[Your Name]
[Your Title]
[Contact Information]
```

### 5. Review for Tone and Clarity

- Read aloud to check flow
- Ensure professional tone
- Remove jargon
- Verify accuracy of information
- Consider cultural sensitivity

### 6. Offer Next Steps

- "Want me to create a version in [Spanish/other language]?"
- "Should I add a parent response section for acknowledgment?"
- "Need a template for follow-up communications?"
- "Shall I schedule this to send at [specific time]?"

## Notes

- Start with positives when possible (even for concerns)
- Focus on partnership, not blame
- Document all communications
- Be mindful of privacy (FERPA)
- Use student's name in greeting
- Proofread before sending

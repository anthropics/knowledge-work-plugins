# Classroom Management

> If you see unfamiliar placeholders or need to check which tools are connected, see [CONNECTORS.md](../CONNECTORS.md).

Develop classroom norms, behavior intervention plans, routines, and management strategies that create a positive, productive learning environment.

## Usage

```
/classroom-management <grade level> <context>
```

Examples:
- `/classroom-management middle-school online`
- `/classroom-management elementary in-person`
- `/classroom-management high-school mixed-hybrid`

## Workflow

### 1. Gather Context

Identify:
- **Grade Level**: Age and developmental stage
- **Setting**: In-person, online, hybrid
- **Class Duration**: Length and frequency
- **Student Needs**: Any special considerations
- **School Context**: Policies, culture

### 2. Create Classroom Norms

Facilitate student co-creation or provide recommendations:

```
## Our Classroom Norms

1. **Respect**
   - Listen when others speak
   - Value diverse perspectives
   - Use kind words and actions

2. **Responsibility**
   - Come prepared each day
   - Complete work to the best of your ability
   - Take care of materials and space

3. **Collaboration**
   - Work well with others
   - Share ideas and resources
   - Support each other's learning

4. **Persistence**
   - Try different strategies when stuck
   - Learn from mistakes
   - Keep trying until successful
```

### 3. Establish Routines

**Entry Routine**
- [ ] Put materials away
- [ ] Begin warm-up/entry ticket
- [ ] Check attendance/attendance tech

**Transition Routines**
- [ ] Materials check
- [ ] Quiet signal/attention cue
- [ ] Group formation procedure

**Exit Routine**
- [ ] Clean-up
- [ ] Reflection/exit ticket
- [ ] Prepare for next class

### 4. Behavior Intervention (if needed)

For students requiring additional support:

```
## Behavior Plan: [Student Name/Description]

### Expected Behavior
- [Specific, observable behavior]

### Support Strategies
1. [Strategy: visual cue, proximity, break card]
2. [Strategy: positive reinforcement, check-in]

### Reinforcement
- [What rewards/privileges for success]
- [What consequences for unmet expectations]

### Monitoring
- [How progress will be tracked]
- [Check-in frequency]
```

### 5. Positive Reinforcement System

Suggest a system appropriate for grade level:

**Elementary**
- Class points/stickers
- Behavior chart
- Reward menu

**Middle/High**
- Class competition
- Responsibility privileges
- Student choice opportunities

### 6. Output Format

Present as:
- Clear section headers
- Bullet points for procedures
- Tables for comparison when helpful
- Copy-paste ready for handouts or LMS

### 7. Offer Next Steps

- "Should I create printable posters of these norms?"
- "Need a digital version for your LMS?"
- "Want a parent communication about classroom expectations?"
- "Shall I create a behavior tracking template?"

## Notes

- Co-create norms with students when possible
- Post visible in classroom (or LMS)
- Teach, model, and practice routines
- Be consistent in enforcement
- Adjust based on class culture

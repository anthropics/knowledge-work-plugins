# Standardized Test Preparation

Strategies for preparing students for standardized assessments while maintaining meaningful instruction.

## Test Structure Overview

### Common Test Formats

| Test Type | Sections | Time | Question Types |
|-----------|----------|------|----------------|
| SAT | Reading, Writing, Math | 3 hours | MC, grid-in |
| ACT | English, Math, Reading, Science | 3 hours 35 min | MC, Essay optional |
| state exams | Varies by subject | 60-90 min per section | MC, short-answer, extended |
| AP Exams | Varies by subject | 2-3 hours | MC, FRQ, essays |

### Question Types Mastery

**Multiple Choice**
- Process of elimination
- Plugging in answers
- Estimation
- Pattern recognition

**Grid-In/Short Answer**
- Unit conversion
- Significant figures
- Multiple solution paths
- Checking work

**Extended Response**
- Planning time
- Clear structure
- Evidence integration
- Time management

## Subject-Specific Strategies

### Math Test Preparation

**Key Skills**
- Formula fluency (know when to use, not just memorize)
- Calculator efficiency
- Estimation techniques
- Mental math shortcuts

**Content Focus**
- Algebra: equations, functions, systems
- Geometry: properties, proofs, coordinate
- Data analysis: graphs, statistics, probability
- Advanced topics: trig, complex numbers

### Reading Test Preparation

**Skills**
- Active reading strategies
- Passage mapping
- Context clues
- Inference drawing

**Question Types**
- Main idea
- Detail comprehension
- Vocabulary in context
- Author's purpose
- Evidence support

### Writing/English Test Preparation

**Skills**
- Grammar rules mastery
- Rhetorical skills
- Organization
- Clarity and concision

**Focus Areas**
- Punctuation
- Sentence structure
- Logical transitions
- Word choice

### Science Test Preparation (AP/State)

**Skills**
- Data interpretation
- Experimental analysis
- Concept application
- Cross-content connections

**Content Areas**
- Biology: cellular, genetics, ecology
- Chemistry: reactions, stoichiometry, periodicity
- Physics: mechanics, electricity, waves
- Environmental science: sustainability, systems

## Practice Test Protocol

### Diagnostic Phase
1. Administer full practice test (timed)
2. Grade with official rubric
3. Analyze by category
4. Identify patterns

### Preparation Phase
1. Focus on weakest areas
2. Teach specific strategies
3. Timed section practice
4. Review mistakes thoroughly

### Final Phase
1. Full practice tests (bi-weekly)
2. Test-taking stamina building
3. Final strategy review
4. Confidence building

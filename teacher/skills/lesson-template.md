# Lesson Template

A comprehensive template for designing standards-aligned lesson plans across subjects and grade levels.

## Essential Components

### 1. Lesson Header

| Field | Purpose |
|-------|---------|
| **Topic** | Main subject content |
| **Grade Level** | Student age/grade |
| **Duration** | Time allocation |
| **Standards** | CCSS or state standards |
| **Learning Objectives** | What students will know/do |
| **Success Criteria** | How we know learning happened |

### 3. Lesson Phase Structure

#### Launch (5-10 minutes)
**Purpose**: Activate prior knowledge, set purpose

- **Hook/Engage**: Attention-grabbing activity
  - Question of the day
  - Quick write
  - Visual analysis
  - Real-world connection

- **Connect**: Link to previous learning
  - Review key concepts
  - Address misconceptions
  - Build schema

- **Objective Share**: Clear learning goal
  - "I Can" statement
  - Success criteria
  - Relevance explanation

#### Explore/Develop (20-30 minutes)
**Purpose**: New learning and guided practice

**Active Learning (15-20 min)**
- Direct instruction (explicit teaching)
- Guided practice (we do together)
- Collaborative work (you do together)
- Differentiation strategies

**Key Elements**
- Think-pair-share
- Group investigations
- Modeling and examples
- Checking for understanding

**Check for Understanding (Every 5-7 minutes)**
- Thumbs up/down
- Whiteboard responses
- Turn-and-talk
- Quick write
- Digital polls

#### Synthesize/Close (10-15 minutes)
**Purpose**: Consolidate learning, assess understanding

- **Summary Activity**
  - Exit ticket
  - Concept map
  - Reflection prompt
  - Peer teaching

- **Review Key Learning**
  - Revisit objectives
  - Connect to big ideas
  - Address remaining questions

- **Preview Next Lesson**
  - Tease upcoming content
  - Connect to current learning
  - Homework assignment

### 5. Differentiation Strategies

**For Advanced Learners**
- Extension activities
- Higher-level questioning
- Student choice
- Acceleration opportunities

**For Struggling Learners**
- Scaffolded support
- Small group instruction
- Sentence starters
- Graphic organizers

**For ELL Students**
- Visual supports
- Sentence frames
- Bilingual resources
- Peer collaboration

**For Students with IEP/504**
- Accommodations list
- Modified tasks
- Assistive technology
- Progress monitoring

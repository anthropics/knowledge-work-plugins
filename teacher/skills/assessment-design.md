# Assessment Design

Guidelines for creating effective, fair, and efficient assessments that accurately measure student learning.

## Assessment Types and When to Use Them

### Formative Assessments (For Learning)

| Type | Best For | Frequency |
|------|----------|-----------|
| Exit Tickets | Quick check at end of lesson | Daily |
| Think-Pair-Share | Activating thinking | Multiple times per lesson |
| Quick Writes | Processing ideas | Weekly |
| Whiteboard Responses | Math/problem solving | Daily |
| Observation Notes | Skills/application | Ongoing |
| Digital Quizzes | Knowledge checks | Weekly |

### Summative Assessments (Of Learning)

| Type | Best For | Weight |
|------|----------|--------|
| Unit Tests | Content mastery | 20-30% |
| Projects | Application/creativity | 20-25% |
| Presentations | Communication skills | 10-15% |
| Portfolios | Growth over time | 15-20% |
| Final Exams | Comprehensive review | 20-30% |

## Question Writing Guidelines

### Multiple Choice

**Do**
- Write clear, concise stems
- Use plausible distractors based on misconceptions
- Keep options similar in length and structure
- Avoid "all of the above" or "none of the above"
- Test higher-order thinking when possible

**Avoid**
- Double negatives
- Grammatical clues to the answer
- Unfamiliar vocabulary in options
- Overly specific trivia

### Short Answer

**Do**
- Ask for specific, unambiguous responses
- Provide clear direction
- Specify format (sentence, phrase, number)
- Use sentence starters if helpful

**Avoid**
- Vague prompts
- Multiple correct answers without specification
- Questions that test reading more than content

### Essay/Performance

**Do**
- Provide clear task description
- Include rubric with criteria
- Specify length/format requirements
- Consider scaffolding options

**Structure**
1. Context/setting
2. Task/task components
3. Constraints/time
4. Submission format

## Rubric Design

### Criteria Selection
- Focus on learning objectives
- 3-5 key criteria max
- Observable, measurable behaviors
- Balance skills and content

### Level Descriptions
| Level | Description |
|-------|-------------|
| **Advanced** | Exceeds expectations, sophisticated thinking |
| **Proficient** | Meets expectations, solid understanding |
| **Developing** | Partial understanding, needs support |
| **Beginning** | Minimal understanding, significant gaps |

### Tips
- Use student-friendly language
- Include exemplars when possible
- Focus on what the work demonstrates
- Avoid negative language

## Scoring and Feedback

### Grading Principles
1. **Consistency**: Same standards for all students
2. **Transparency**: Rubrics shared in advance
3. **Timeliness**: Return within 1-2 class days
4. **Actionable**: Specific, improvement-focused feedback

### Feedback Matrix
| Level | Feedback Focus |
|-------|----------------|
| Advanced | Extension, deeper analysis, connections |
| Proficient | Refinement, polish, next steps |
| Developing | Targeted support, re-teaching |
| Beginning | Foundational concepts, basic understanding |

## Academic Integrity

### Prevention
- Multiple assessment versions
- Open-ended questions
- Authentic, relevant tasks
- Process documentation

### Detection
- Unusual vocabulary shifts
- Inconsistent quality
- Format inconsistencies
- Suspicious timing

### Response Protocol
1. Document observations
2. Review with department/leadership
3. Contact student for conference
4. Apply school integrity policy
5. Document resolution

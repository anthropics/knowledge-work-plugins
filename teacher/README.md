# Teacher Plugin

A plugin that helps teachers plan lessons, create assessments, provide feedback, and communicate with families. Connects to educational tools like Google Classroom, Canvas, Moodle, and more.

## What This Plugin Does

This plugin helps you:

- **Plan lessons** with standards-aligned lesson plans that include objectives, activities, and assessments
- **Create assessments** with quizzes, tests, and assignments complete with answer keys
- **Provide feedback** that is specific, actionable, and growth-oriented
- **Manage classroom** with norms, routines, and behavior intervention plans
- **Communicate with families** through professional, positive messages

## Slash Commands

| Command | Purpose |
|---------|---------|
| `/lesson-plan` | Create a comprehensive lesson plan |
| `/assessment` | Create quizzes, tests, or assignments |
| `/feedback` | Provide detailed feedback on student work |
| `/classroom-management` | Create classroom norms and routines |
| `/parent-communication` | Draft messages to families |

## Connectors

This plugin connects to educational tools:

- **Google Classroom** - Class management and assignments
- **Canvas** - Course management
- **Moodle** - Course platform
- **Google Drive** - File storage
- **Gmail** - Email communication
- **Google Calendar** - Scheduling
- **Notion** - Organization
- **Slack** - Team collaboration

## Getting Started

1. Install this plugin in Claude Code or Claude Cowork
2. Connect your educational tools via the plugin settings
3. Try a command like `/lesson-plan photosynthesis 7th-grade 45-minutes`

## Customizing for Your School

These templates are generic starting points. Make them yours:

1. **Add your curriculum** - Include your scope and sequences
2. **Customize rubrics** - Use your school's grading criteria
3. **Add school policies** - Include your discipline and feedback policies
4. **Save templates** - Save frequently used lesson plans and assessments

## Tips for Best Results

- Be specific about grade level and content
- Include standards when available
- Share context about your students
- Ask for variations if the first draft isn't quite right

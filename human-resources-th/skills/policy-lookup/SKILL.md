# ค้นหานโยบายบริษัท (Policy Lookup)

ค้นหาและอธิบายนโยบายบริษัทอย่างตรงไปตรงมา

## วิธีที่ใช้ได้

```
┌─────────────────────────────────────────────────────────────────┐
│                    ค้นหานโยบาย                                   │
├─────────────────────────────────────────────────────────────────┤
│  ใช้ได้ทันที (ไม่ต้องเชื่อมต่อระบบ)                            │
│  ✓ ถามนโยบายใดก็ได้ในภาษาธรรมดา                              │
│  ✓ วาง Employee Handbook มาและจะค้นหาให้                       │
│  ✓ ได้คำตอบชัดเจน ไม่มีศัพท์เทคนิค                            │
├─────────────────────────────────────────────────────────────────┤
│  ทรงพลังขึ้น (เมื่อเชื่อมต่อระบบ)                              │
│  + Knowledge Base: ค้นหาเอกสาร Handbook อัตโนมัติ             │
│  + HRIS: ดึงข้อมูลเฉพาะพนักงาน (วันลาคงเหลือ, สวัสดิการ)     │
└─────────────────────────────────────────────────────────────────┘
```

## หัวข้อนโยบายทั่วไป

- **วันลา**: ลาพักร้อน, ลาป่วย, ลาคลอด/ลาดูแลบุตร, ลากิจ, ลาเนื่องจากเสียชีวิต
- **สวัสดิการ**: ประกันสุขภาพ, ประกันชีวิต, กองทุนสำรองเลี้ยงชีพ, ประกันสังคม
- **ค่าตอบแทน**: รอบจ่ายเงินเดือน, โบนัส, การ Vest หุ้น, การเบิกค่าใช้จ่าย
- **Work from Home**: นโยบาย WFH, ทำงานต่างจังหวัด/ต่างประเทศ, เบี้ยอุปกรณ์
- **การเดินทาง**: จองตั๋ว, เบี้ยเลี้ยง, การเบิกค่าใช้จ่าย, กระบวนการอนุมัติ
- **จรรยาบรรณ**: ประมวลจรรยาบรรณ, นโยบายป้องกันการคุกคาม, ผลประโยชน์ทับซ้อน
- **การพัฒนา**: งบพัฒนาทักษะ, นโยบายเข้าร่วม Conference, ทุนการศึกษา

## วิธีตอบ

1. ค้นหา ~~knowledge base สำหรับเอกสารนโยบายที่เกี่ยวข้อง
2. ให้คำตอบชัดเจนในภาษาธรรมดา
3. อ้างข้อความนโยบายโดยตรง
4. ระบุข้อยกเว้นหรือกรณีพิเศษ
5. บอกว่าควรติดต่อใครหากมีกรณีที่ซับซ้อน

**ข้อควรระวัง:**
- อ้างเอกสารต้นทางและหัวข้อเสมอ
- หากไม่พบนโยบาย ให้บอกตรงๆ อย่าเดา
- สำหรับคำถามกฎหมายหรือการปฏิบัติตามกฎระเบียบ แนะนำให้ปรึกษา HR หรือที่ปรึกษากฎหมาย

## ผลลัพธ์

```markdown
## นโยบาย: [หัวข้อ]

### คำตอบสั้น
[คำตอบตรงๆ 1-2 ประโยค]

### รายละเอียด
[รายละเอียดนโยบายอธิบายในภาษาธรรมดา]

### ข้อยกเว้น / กรณีพิเศษ
[ข้อยกเว้นหรือกรณีพิเศษที่เกี่ยวข้อง]

### ผู้ติดต่อ
[บุคคลหรือทีมสำหรับคำถามที่นอกเหนือเอกสาร]

### แหล่งที่มา
[เอกสาร, หัวข้อ หรือหน้าที่ดึงข้อมูลมา]
```

## สิทธิ์ตามกฎหมายแรงงานไทย (ขั้นต่ำตามกฎหมาย)

| ประเภทการลา | สิทธิ์ขั้นต่ำ | หมายเหตุ |
|------------|-------------|---------|
| ลาพักร้อน | 6 วัน/ปี (หลังทำงานครบ 1 ปี) | บริษัทส่วนใหญ่ให้มากกว่า |
| ลาป่วย | 30 วัน/ปี (ได้รับเงินเดือน) | เกิน 3 วันต้องมีใบรับรองแพทย์ |
| ลาคลอด | 98 วัน (ได้รับเงิน 45 วัน) | นายจ้างจ่าย 45 วัน + ประกันสังคม 45 วัน |
| ลากิจ | 3 วัน/ปี | บริษัทกำหนดเพิ่มเติมได้ |
| ลาอุปสมบท | 120 วัน (เฉพาะชาย) | ได้รับเงินเดือน |
| วันหยุดนักขัตฤกษ์ | 13 วัน/ปี | ตามประกาศของรัฐบาล |

## หากเชื่อมต่อกับระบบ

หากเชื่อมต่อกับ **~~knowledge base**:
- ค้นหา Employee Handbook และเอกสารนโยบายอัตโนมัติ
- อ้างเอกสาร หัวข้อ และหน้าที่เฉพาะ

หากเชื่อมต่อกับ **~~HRIS**:
- ดึงข้อมูลเฉพาะพนักงาน เช่น วันลาคงเหลือ, สวัสดิการที่เลือก และสถานะการสมัคร

## เคล็ดลับ

1. **ถามในภาษาธรรมดา** — "ลาพักร้อนไปต่างประเทศ 1 เดือนได้ไหม" ดีกว่า "นโยบาย International Remote Work"
2. **ระบุให้ชัดเจน** — "ลาพักร้อนสำหรับพนักงาน Part-time" ได้คำตอบดีกว่า "นโยบายลาพักร้อน"

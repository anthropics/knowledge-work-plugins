# สรรหาบุคลากร (Recruiting Pipeline)

ช่วยจัดการกระบวนการสรรหาตั้งแต่ค้นหาผู้สมัครจนถึงรับข้อเสนองาน

## ขั้นตอน Pipeline

| ขั้นตอน | คำอธิบาย | การดำเนินการหลัก |
|---------|-----------|-----------------|
| ค้นหา | ระบุและติดต่อผู้สมัคร | ส่งข้อความเชิญ |
| คัดกรองเบื้องต้น | โทรศัพท์/วิดีโอสัมภาษณ์ | ประเมินความเหมาะสมเบื้องต้น |
| สัมภาษณ์ | สัมภาษณ์กับทีม | ประเมินอย่างมีโครงสร้าง |
| สรุปผล | ทีมตัดสินใจ | ประเมินและเปรียบเทียบ |
| ยื่นข้อเสนอ | ส่ง Offer Letter | แพ็กเกจค่าตอบแทน, การเจรจา |
| ตอบรับ | ผู้สมัครตอบรับ | ส่งต่อให้กระบวนการรับเข้าทำงาน |

## ตัวชี้วัดสำคัญ

- **ความเร็วแต่ละขั้นตอน**: จำนวนวันในแต่ละ Stage
- **อัตราการผ่านแต่ละขั้นตอน**: Drop-off ระหว่าง Stage
- **ประสิทธิภาพช่องทาง**: ช่องทางไหนได้ผู้สมัครที่ผ่านการคัดเลือก
- **อัตราตอบรับ Offer**: Offer ที่ส่งเทียบกับที่ตอบรับ
- **ระยะเวลาบรรจุ**: นับจากวันเปิดรับถึงวันตอบรับ

## หากเชื่อมต่อกับระบบ ATS

ดึงข้อมูลผู้สมัครอัตโนมัติ อัปเดตสถานะ และติดตาม Pipeline Metrics แบบ Real-time

## บริบทกฎหมายแรงงานไทย

- การประกาศรับสมัครงานต้องไม่มีการเลือกปฏิบัติตาม พ.ร.บ. คุ้มครองแรงงาน
- ห้ามระบุเพศ อายุ (ยกเว้นที่กฎหมายกำหนด) หรือสถานภาพในประกาศงาน
- การทดสอบฝีมือ/ข้อเขียนถือเป็นส่วนหนึ่งของกระบวนการสรรหาที่ถูกต้องตามกฎหมาย

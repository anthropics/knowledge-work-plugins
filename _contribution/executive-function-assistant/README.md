# Executive Function Assistant

Turn Claude into an automatic productivity coach that detects your specific blocker and applies the right intervention — without asking you to pick a mode.

## What it does

This skill diagnoses the root cause of your current block and responds directly with the right action:

| Blocker detected | Mode activated |
|---|---|
| Obvious exhaustion | **Energy Management** — stops the productivity machine, forces rest first |
| Too many open tasks | **Chaos Triage** — picks ONE task, ignores the rest ruthlessly |
| Stuck at the start | **Break Paralysis** — 3 ridiculously small steps (<60 seconds each) |
| Time blindness | **Time Auditor** — realistic map including hidden subtasks |
| No accountability | **Body Double** — atomic goal + check-in message |
| Context switching | **Context Reset** — 3-minute mental cleaner routine |
| Boring/repetitive task | **Gamification** — converts the task into a mission based on your interests |
| Dev task too big | **Dev Focus** — exact file → method → minimal change → compile |
| Perfectionism loop | **Anti-Perfectionism** — defines the minimum viable deliverable and what stays "ugly" on purpose |

## Key behavior

**Never asks you to choose a mode.** The skill reads your message, diagnoses internally, and responds in the correct mode immediately — no menus, no prompts.

**Energy first (Rule 0).** If there are signs of severe burnout, all productivity tools stop. Rest is the only valid output.

## Designed for

Developers and analysts who deal with ADHD-adjacent blockers: task paralysis, context overload, time blindness, and the perfectionism–procrastination cycle.

## No external tools required

This skill works standalone with no MCP connections needed.
